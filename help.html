reclaim your disk space

SpaceSniffer Portable is the SpaceSniffer lost disk space tool packaged with a PortableApps.com launcher as a portable app, so you can analyze lost disk space on your iPod, USB flash drive, portable hard drive, etc. It has all the same features as SpaceSniffer, plus, it leaves no personal information behind on the machine you run it on, so you can take it with you wherever you go.
